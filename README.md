# Django-Settings(장고 환경분리)
(Django Settings for Local and Production Development)
<br>
<br>


>**환경을 분리하는 이유**
<center>
개발 환경에서는 DEBUG = True, 실제 운영 환경에서 DEBUG = Flase로 변경하여 작업을 진행함.<br>
Django 프로젝트 진행 시 개발 환경과 실제 운영환경에 대해서 분리가 필요함.<br>
</center><br>
